# Smart Agriculture System
Hệ thống Nông nghiệp Thông minh ứng dụng IoT để giám sát độ ẩm đất, nhiệt độ, độ ẩm không khí và mực nước theo thời gian thực. Dữ liệu được gửi từ ESP32 qua MQTT đến backend để lưu trữ và xử lý. Hệ thống hỗ trợ tưới tự động, cảnh báo khi vượt ngưỡng và quản lý thiết bị thông qua giao diện web.
